# Temperature Converter (Java Swing)

A simple **Temperature Converter Desktop Application** built using **Java Swing**.  
This application allows users to convert temperature values between **Celsius, Fahrenheit, and Kelvin** through an interactive graphical interface.

---

# Project Overview

The **Temperature Converter** is a GUI-based Java application that takes a temperature input from the user and converts it into the other temperature units.

The program supports conversions between:

- Celsius
- Fahrenheit
- Kelvin

It demonstrates **Java GUI development, event handling, and mathematical calculations**.

---

# Features

- 🌡️ Convert temperature between **Celsius, Fahrenheit, and Kelvin**
- 🖥️ User-friendly **Graphical User Interface**
- 📥 Input temperature using a text field
- 🔽 Select temperature unit using dropdown menu
- 🔄 Instant conversion using a button
- ⚠️ Error handling for invalid input
- 📊 Displays all converted values simultaneously

---

# Technologies Used

- Java
- Java Swing
- AWT
- Event Handling (ActionListener)
- Basic Mathematical Operations
